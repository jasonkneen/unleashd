Unleashd

A local, git-worktree multi-agent swarm orchestrator for Codex, Claude, Gemini, and OpenCode.

git clone --recursive https://github.com/nbardy/unleashd && cd unleashd && pnpm install && pnpm build && pnpm start

Visual Swarm Debugger

Monitor your agents in real-time. See their prompts, tool calls, and state transitions via a clean Web UI.

Git Worktrees

Every agent works in an isolated Git worktree, preventing collisions and making merges seamless.

Multi-Model Support

Route tasks between Claude, Codex, Gemini, and OpenCode instances within the same swarm.
