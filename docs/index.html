Unleashd is an open source SWARM FIRST ADE (Agent Development Environment) with swarms as first-class primitives. It lets you bring your own max accounts, reduce costs, and manage agents and swarms across multiple providers. Supports Claude, Codex, OpenCode, and Gemini CLI, and lets you filter and search chats across all platforms while launching swarms that coordinate agents across environments.

Keep your token costs down, utilize the best existing harnesses for each model AND run your oompa loompa swarms for autonomous swarm development, built on top of the oompa loompa.

Visual Swarm Debugger

Monitor your agents in real-time. See their prompts, tool calls, and state transitions via a clean, unified Web UI.

Git Worktrees

Every agent works in an isolated Git worktree, preventing collisions and making merges completely seamless.

Multi-Model Support

Route tasks dynamically between Claude, Codex, Gemini, and OpenCode instances within the same unified swarm.
